CSS membuat tooltip dasar

Buat tooltip yna gmuncul saat pengguna menggerakkan crsor di atas elemen

Contoh

Membuat tooltip dasar:

hover me lebih banyak teks

contoh yang dijelaskan

HTML:

  • Gunakan elemen wadah (seperti <div>) dan tambahkan kelas "tooltip". Ketika mouse pengguna di atas <div> ini, itu akan menunjukkan teks tooltip.
  • Teks tooltip ditempatkan di dalam elemen inline(seperti <span>) dengan class="text".

CSS:

  • Bagian class tooltip menggunakan position= relative;, yang diperlukan untuk memposisikan class text secara (position: absolute;). Tip: lihat contoh di bawah ini tentang cara memposisikan tooltip.
  • Bagian class text memegang bagian text tooltip ketika di hover. Hal ini tersembunyi secara default, dan akan terlihat pada saat di hover.
  • Bagian :hover digunakan untuk menampilkan teks tooltip ketika pengguna mengarahkan cursor ke atas class tooltip.

position tooltip

Anda dapat memposisikan tooltip yang anda suka. Disini kita akan menunjukkan bagaimana untuk posisi tooltip ke kiri, kanan, atas dan bawah.

tooltip kanan dan kiri blok

Dalam contoh ini, tooltip ditempatkan di sebelah kanan (left:105%;) ketika meng-hover class tooltip. Juka mencatat bahwa top: -5px; digunakan untuk menempatkan di tengah-tengah elemen wadahnya. Kami menggunakan -5px dikarenakan Teks memiliki top dan bottom padding 5px. Jika anda meningkatkan padding-nya, serta meningkattkan nilai top properti untuk pastikan bahwa itu tetap ditengah (jika ini adalah sesuatu yang anda inginkan). Hal yang sama berlaku jika anda ingin tooltip ditempatkan di sebelah kiri.
